  1. Open VS Code terminal
  2. Select "Claude Code" from dropdown
  3. Keep "npm run dev" running
  4. Ask Claude Code to modify the app:

     Example:
     "Change the edit panel color to purple"
     "Add a priority field to goals"
     "Make the save button green"

  Claude Code will update the files and changes appear live!
